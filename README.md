# CSE-4104-7A-T02

**Project Description**
An intelligent, full-stack web application that transforms raw business sales data into real-time, AI-driven insights. The system enables business managers and analysts to monitor revenue trends, forecast future sales, detect anomalies, and interact with their data through a natural language chatbot — all without requiring any data science expertise.

**Project Title**

AI Sales Analytics Dashboard: An Intelligent, Data-Driven Business Intelligence System


**Proposed Features**

Authentication & User Management


**Register, login, and logout**

JWT-based authentication with bcrypt password hashing

Role-based access control: Admin, Manager, Viewer

User profile management



**Sales Data Management**

Add, edit, and delete sales records manually

Bulk CSV file upload for large datasets

Filter sales by date, product, region, and category

Search functionality across all records


**Dashboard & Analytics**

KPI summary cards: total revenue, total orders, average order value

Revenue over time — interactive line chart

Sales by product and category — bar chart

Sales by region — pie and donut chart

Monthly comparison view

Top performing products table


**AI Features**

AI Insight Generator — analyses current sales data and produces a written business summary

AI Sales Forecaster — predicts next-month revenue using scikit-learn regression models

AI Chatbot — allows users to query sales data using natural language questions

AI Anomaly Detector — automatically flags unusual spikes or drops in sales data

AI Recommendation Engine — suggests actionable steps such as restocking or running promotions



**Reports & Export**

AI-generated monthly sales report

Export dashboard data as CSV

Export reports as PDF

Low-performance alerts and notifications
